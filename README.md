## 📖 Descripción
Este proyecto es la **Landing Page oficial de Soza Systems**, una agencia de desarrollo web especializada en sitios de alto impacto visual y rendimiento.
El diseño sigue una estética **Dark Mode con acentos Neón** (Cyan, Magenta, Verde) para transmitir modernidad, tecnología y energía, pero con un enfoque estratégico para la conversión.

### Características Principales y Técnicas:
*   ✅ **Estética Premium:** Interfaz oscura con efectos de brillo (glow) y gradientes vibrantes sin perder legibilidad.
*   ✅ **Diseño Responsivo:** Adaptación fluida para móviles, tablets y escritorio.
*   ✅ **Código Limpio:** Uso de React Hooks, componentes modulares y CSS Variables.
*   ✅ **Navegación Suave:** Scroll animado entre secciones.
*   ✅ **Datos Separados:** Fácil edición de servicios y clientes desde un archivo de configuración.
*   ✅ **Calidad de Código:** Configuración estricta de **ESLint** para mantener el código limpio y libre de errores.
